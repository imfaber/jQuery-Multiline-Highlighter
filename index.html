Usage:

Create an element and style it as you wish


h2.highlight{
	padding: 5px 20px;
	background: orange;
	width: 200px;
}

...

<h2 class="highlight">Lorem ipsum dolor sit...</h2>

Include jQuery and Multiline Highlither plugin


<script src="http://code.jquery.com/jquery-1.8.1.min.js"></script>
<script src="js/jquery.multilineHighlighter.js"></script>

Initialize Multiline Highlither


$(document).ready(function(){
	$('.highlight').multilineHighlighter();
});
